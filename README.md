# IITC Plugin: Portal Details External Window

ポータルの詳細情報（`#portaldetails`）を、IITCのメイン画面から分離して別ウィンドウで**常時表示**し、メイン画面でのマップ視認性を最大化するためのプラグインです。

## 特徴

- **IITCの完全なレイアウト再現**: 親ウィンドウのスタイルシートを自動的に引き継ぐため、レゾネーターやMODの配置、画像プレビューが崩れることなく綺麗に表示されます。
- **常時表示仕様**: プラグイン起動時やポータル選択時に自動でウィンドウが起動します。誤って閉じてしまった場合でも、次のポータルを選択した際に自動で復帰します。
- **外部プラグインの動的更新に対応**: `MutationObserver` を用いて親ウィンドウのDOM変更を監視しているため、非同期通信などで後から遅延表示される他のプラグインの情報も、瞬時に子ウィンドウへ同期されます。
- **操作のリアルタイム連動**: 子ウィンドウ側で行ったボタンのクリックや、テキストボックス・チェックボックス等への入力操作は、即座に親ウィンドウ（IITC本体）へ伝播します。これにより、子ウィンドウからでも外部プラグインの各種操作がシームレスに行えます。
- **画面の有効活用**: メインのマップ画面からは詳細パネルが完全に非表示となり、視認領域が最大化されます。デュアルディスプレイ環境での利用に最適化されています。

## インストール方法

1.  Tampermonkey等のユーザースクリプトマネージャーに、本スクリプトを追加して保存します。
2.  Intel Mapをリロードします。

## 使い方・注意事項

- **【重要】ポップアップブロックの解除**: 本プラグインは自動的に新しいウィンドウ（ポップアップ）を開く性質上、初回起動時にブラウザの標準機能によってブロックされることがほとんどです。
  - アドレスバー右側に表示される警告アイコン等から、**「ingress.com のポップアップを常に許可する」**設定を行ってください。
  - もしブロックされて自動で開かない場合は、左下のツールボックスにある「詳細ウィンドウ強制表示」リンクを手動でクリックしてください。
- 子ウィンドウ側でのテキスト入力中、裏側で画面の同期（HTMLの上書き）が発生しても、自動的にカーソル位置とフォーカスが復元されるため、タイピングが途切れることはありません。
