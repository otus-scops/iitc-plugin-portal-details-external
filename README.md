# IITC Plugin: Portal Details External Window

ポータルの詳細情報（`#portaldetails`）を、IITCのメイン画面から分離して別ウィンドウで表示し、メイン画面でのマップ視認性を最大化するためのプラグインです。

## 特徴

- **直感的なアイコン切り替え**: ポータル詳細のタイトルバー（右上の閉じるボタンの隣）に追加されるアイコンをクリックするだけで、標準表示とポップアウト（別ウィンドウ）表示を瞬時に切り替えられます。
- **双方向の表示連携**: 子ウィンドウ側で切り替えアイコンを押したり、手動で子ウィンドウを閉じたりした場合でも、自動的に親ウィンドウの表示が標準モードに復帰します。
- **状態に応じたアイコン変化**: ポップアウト中はアイコンが反転し、「標準表示に戻る（矢印が内側を向く）」デザインへダイナミックに切り替わります。
- **IITCの完全なレイアウト再現**: 親ウィンドウのスタイルシートを自動的に引き継ぐため、レゾネーターやMODの配置、画像プレビューが崩れることなく綺麗に表示されます。
- **外部プラグインの動的更新に対応**: `MutationObserver` を用いて親ウィンドウのDOM変更を監視しているため、非同期通信などで後から遅延表示される他のプラグインの情報も、瞬時に子ウィンドウへ同期されます。
- **操作のリアルタイム連動**: 子ウィンドウ側で行ったボタンのクリックや、テキストボックス・チェックボックス等への入力操作は、即座に親ウィンドウ（IITC本体）へ伝播します。

## 使い方・注意事項

1.  インストール後、ポータルを選択すると、詳細パネルのタイトルバー右上に「ポップアウト切り替えアイコン」が表示されます。
2.  このアイコンをクリックすることで、表示モードを切り替えることができます。
    - **ON**: マップ上の詳細パネルが消え、専用の子ウィンドウが立ち上がります。
    - **OFF**: 子ウィンドウが閉じ、IITCの標準の挙動（サイドバーに詳細が表示される状態）に戻ります。
3.  **【重要】ポップアップブロックの解除**: 本プラグインは自動的に新しいウィンドウ（ポップアップ）を開く性質上、初回起動時にブラウザの標準機能によってブロックされることがほとんどです。
    - アドレスバー右側に表示される警告アイコン等から、**「ingress.com のポップアップを常に許可する」**設定を行ってください。
